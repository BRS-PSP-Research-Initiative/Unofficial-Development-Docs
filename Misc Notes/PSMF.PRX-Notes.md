These notes are primarily useful for showing various pieces of PSP specific information (such as SDK version, possibly hardware developed on, etc). [This](https://www.psdevwiki.com/psp/PRX_File_Format) resource was used extensively to help parse the file. This is also mostly from the Japanese release of the game (except where noted otherwise).

---

* 0x04 - Module Priviledge Level - 6
* 0x06 - Compression Type - 0100 - 2RLZ
* 0x08 - 0x09 - Module Version 1.0.1
* 0x26 - Shows Module Version 2.0.1 (?)
* 0x78 - SDK Version - 5.1.0
* 0x7C - Encrypte Type - 6.0.3 - Currently Unknown Encryption Type
* 0x7E - Overlap Size - 4

* Unknown if PSP Header V1 or V2 at this time
