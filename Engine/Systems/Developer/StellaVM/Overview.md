This is a general overview of what is known about StellaVM.

* Purpose: General purpose Virtual Machine used primarily for handling Tasks.
* Duties:
    * Read and execute byte-compiled:
        * Stage 1 Heap Allocation Tasks
        * Memory Card Save Tasks
        * Memory Card Game Installation Tasks
* Observed Specifications:
    * At least one usage suggests that memory block size uses at least 4KB of Heap Memory to perform Memory Card related Tasks
